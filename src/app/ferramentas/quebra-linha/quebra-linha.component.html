TEXT

Online Text Cleaner and Line Editor

Clean and transform plain text with quick operations for line breaks, replacements, ordering and repeated content. The editor is designed for preparing lists, logs, copied spreadsheet values, code snippets and bulk text.

Use the available transformations to normalize inconsistent input before importing it into another application or comparing it with a reference document.

Instructions

Remove Breaks

Paste the text and click this button to turn the pasted content into a single line.

Trim

Removes line breaks, duplicate spaces and tab characters.

Break by Character

Inserts a line break after the specified character.

Break by Quantity

Inserts a line break every n characters.

Replace

Replaces the text from the first input with the text from the second input.

Uppercase

Converts all letters in the text to uppercase.

Lowercase

Converts all letters in the text to lowercase.

Title Case

Capitalizes the first letter of each word.

Number Lines

Adds ordered numbers at the beginning of each line.
