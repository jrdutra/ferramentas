Como usar

Digite ou cole qualquer texto no campo acima. Os hashes são calculados automaticamente em tempo real usando MD5, SHA-1, SHA-256 e SHA-512.

Sobre os algoritmos

MD5 — Produz um hash de 128 bits (32 caracteres hex). Amplamente usado para verificar integridade de arquivos, mas não recomendado para segurança criptográfica.

SHA-1 — Produz 160 bits (40 caracteres hex). Historicamente popular, mas considerado fraco para uso criptográfico moderno.

SHA-256 — Parte da família SHA-2. Produz 256 bits (64 caracteres hex). Padrão amplamente adotado para segurança em geral.

SHA-512 — Também da família SHA-2. Produz 512 bits (128 caracteres hex). Oferece maior margem de segurança que o SHA-256.
