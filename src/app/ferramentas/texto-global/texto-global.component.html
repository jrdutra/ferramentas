Instructions

Text Sharer lets you synchronize text in real time across multiple screens, without registration or login. Define a Group and a Channel, and anyone using the same pair will see changes instantly.

How to use

Set Group and Channel

Type a group name and a channel name in the fields on the left. As soon as both are filled in, you automatically join that channel. Any text typed in the box is transmitted in real time to all other screens connected to the same group and channel.

Share the URL

When group and channel are filled in, a shareable URL appears below the fields. Click the 📋 icon to copy it. When this URL is opened on another device or tab, the group and channel are already filled in and the connection is established automatically.

Browse the Directory

The right column displays all groups and channels with active connections right now. Select a group in the combo box and use the search field to filter channels. Click any channel to join it and load the current text.

Change History

The center column records the time of each significant change to the channel text. A new entry is created after 10 seconds without edits, marking the moment when that version was considered stable. History is kept only while the channel is active.

Group Administrator

The first screen to enter a group automatically becomes the administrator (ADM). The administrator can delete channels individually - other screens in that channel automatically move to an adjacent channel - and can also delete the entire group, which notifies all connected screens. If the administrator leaves, the next oldest member inherits the role.

Attention: a channel's text and history are kept in memory for up to 5 minutes after the last screen disconnects. After that, the channel is closed and the data is permanently discarded. There is no database persistence.

Groups and Channels

A group is a workspace that contains multiple channels. Use groups to separate projects, teams or different contexts. A channel is the sharing unit: each channel has its own independent text and history.

Any group and channel combination is valid - there is no prior registration. The channel starts existing as soon as the first screen enters it and stops existing 5 minutes after the last screen leaves.
