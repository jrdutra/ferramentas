Instructions

How to split a PDF into individual pages

Click the upload area and select the PDF file you want to split. The tool automatically detects the number of pages and displays that information on screen. Then choose the desired output format by clicking one of the two download buttons.

Download ZIP with PDFs

When you click "Download ZIP with PDFs", each page of the original document is extracted as an independent .pdf file. The files are named sequentially (page-001.pdf, page-002.pdf, etc.) and compressed into a single .zip file for easier download.

Download ZIP with JPGs

When you click "Download ZIP with JPGs", each page is rendered as a .jpg image at 2x scale (high resolution) using the PDF.js library. The images are compressed into a .zip file and named with the same sequence (page-001.jpg, page-002.jpg, etc.). This option is useful for using pages in presentations, image editors or other contexts that do not support PDF.

Privacy

All processing happens entirely in your browser. No file is sent to external servers - the PDF is read and processed locally on your machine.

What is ZIP

ZIP is a compressed file format that groups multiple files into a single package. It is widely supported by all operating systems: on Windows, right-click and choose "Extract all"; on macOS, double-click the .zip file to decompress its contents.
