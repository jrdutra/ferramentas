Instruções

Como separar um PDF em páginas individuais

Clique na área de upload e selecione o arquivo PDF que deseja separar. A ferramenta detecta automaticamente o número de páginas e exibe essa informação na tela. Em seguida, escolha o formato de saída desejado clicando em um dos dois botões de download.

Baixar ZIP com PDFs

Ao clicar em "Baixar ZIP com PDFs", cada página do documento original é extraída como um arquivo .pdf independente. Os arquivos são nomeados sequencialmente (pagina-001.pdf, pagina-002.pdf, etc.) e compactados em um único arquivo .zip para facilitar o download.

Baixar ZIP com JPGs

Ao clicar em "Baixar ZIP com JPGs", cada página é renderizada como uma imagem .jpg em escala 2× (alta resolução) usando a biblioteca PDF.js. As imagens são compactadas em um arquivo .zip nomeadas da mesma forma sequencial (pagina-001.jpg, pagina-002.jpg, etc.). Esta opção é útil para usar as páginas em apresentações, editores de imagem ou outros contextos que não suportam PDF.

Privacidade

Todo o processamento acontece inteiramente no seu navegador. Nenhum arquivo é enviado a servidores externos — o PDF é lido e processado localmente na sua máquina.

O que é ZIP

ZIP é um formato de arquivo compactado que agrupa múltiplos arquivos em um único pacote. É amplamente suportado por todos os sistemas operacionais: no Windows basta clicar com o botão direito e escolher "Extrair tudo", no macOS basta dar duplo clique no arquivo .zip para descompactar o conteúdo.
