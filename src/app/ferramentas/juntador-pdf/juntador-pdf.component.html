Instructions

How to merge PDFs and images

Click the upload area or drag files directly onto it. You can load PDFs and images in any format (JPG, PNG, GIF, WebP, BMP) at the same time and in any combination. After loading, the files appear in an ordered list. Click "Merge and Download PDF" to generate the final document.

How to reorder files

Drag any item in the list by the handle (six-dot icon on the left) and drop it in the desired position. The order shown in the list is exactly the page order in the final PDF. To remove a specific file, click the "X" icon on the right side of the item. To remove all files at once, use the "Clear all" button.

Download as Image

The "Download as Image" button stacks all loaded files (PDFs and images) vertically into a single JPG image. Each PDF page and each image occupies a horizontal band, and the final width is the largest width among all items - the others are scaled proportionally. The result is downloaded as a single .jpg file.

Supported formats and privacy

The merger natively supports JPEG and PNG, embedding them directly into the PDF. Other formats (GIF, WebP, BMP and similar) are automatically converted to JPEG via canvas before being inserted. All processing happens entirely in your browser - no file is sent to external servers.

What is a PDF

PDF (Portable Document Format) is a format created by Adobe that preserves a document's formatting regardless of the operating system or application used to view it. It is the standard for sharing documents, contracts and reports that need to keep their original appearance across different devices.

The merge performed by this tool uses the pdf-lib library, which reads the pages from each input file and copies them into a new PDF document while preserving the original resolution, fonts and layout of each page.
