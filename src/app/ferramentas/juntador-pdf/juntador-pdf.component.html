Instruções

Como juntar PDFs e imagens

Clique na área de upload ou arraste arquivos diretamente para ela. Você pode carregar PDFs e imagens de qualquer formato (JPG, PNG, GIF, WebP, BMP) ao mesmo tempo e em qualquer combinação. Após carregar os arquivos, eles aparecem em uma lista ordenada. Clique em "Juntar e Baixar PDF" para gerar o documento final.

Como reordenar os arquivos

Arraste qualquer item da lista pela alça (ícone de seis pontos à esquerda) e solte na posição desejada. A ordem exibida na lista é exatamente a ordem das páginas no PDF final. Para remover um arquivo específico, clique no ícone "X" à direita do item. Para remover todos de uma vez, use o botão "Limpar tudo".

Baixar como Imagem

O botão "Baixar como Imagem" empilha verticalmente todos os arquivos carregados (PDFs e imagens) em uma única imagem JPG. Cada página de PDF e cada imagem ocupa uma faixa horizontal, e a largura final é a maior largura entre todos os itens — os demais são escalados proporcionalmente. O resultado é baixado como um único arquivo .jpg.

Formatos suportados e privacidade

O Juntador suporta nativamente JPEG e PNG, incorporados diretamente ao PDF. Outros formatos (GIF, WebP, BMP e similares) são convertidos automaticamente para JPEG via canvas antes de serem inseridos. Todo o processamento acontece inteiramente no seu navegador — nenhum arquivo é enviado a servidores externos.

O que é um PDF

PDF (Portable Document Format) é um formato criado pela Adobe que preserva a formatação de um documento independentemente do sistema operacional ou aplicativo usado para visualizá-lo. É o padrão para compartilhamento de documentos, contratos e relatórios que precisam manter sua aparência original em diferentes dispositivos.

A junção realizada por esta ferramenta usa a biblioteca pdf-lib, que lê as páginas de cada arquivo de entrada e as copia para um novo documento PDF mantendo resolução, fontes e layout originais de cada página.
