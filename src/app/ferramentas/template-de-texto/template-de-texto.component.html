Load Template

Paste the text template in the text field. Variables must be indicated as [[variable]]. Click this button to identify the variables and generate dedicated input fields for each one. Shows the original text template pasted into the textarea, without substitutions. Useful for reviewing the template before generating the final text.

Text Field (Textarea)

Use this field to paste the text template. Insert variables in the [[variable]] format so they can be identified and processed by the tool. This template is used as the base for generating customized texts.

Purpose

This tool lets you create standardized texts with customized snippets quickly and safely, automatically replacing the variables indicated in the template with the values entered in the generated fields.
