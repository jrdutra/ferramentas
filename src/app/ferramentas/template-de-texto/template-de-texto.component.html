Instruções

Carregar Template

Cole o template de texto no campo de texto. As variáveis devem estar indicadas como [[variavel]]. Clique neste botão para identificar as variáveis e gerar campos de entrada exclusivos para cada uma. Após preencher os valores nos campos gerados, esses valores serão automaticamente substituídos no template onde as variáveis aparecem.

Visualizar Template

Exibe o template de texto original que foi colado no campo de texto textarea, sem as substituições. Útil para revisar o template antes de gerar o texto final.

Campo de Texto (Textarea)

Use este campo para colar o template de texto. Insira as variáveis no formato [[variavel]] para que sejam identificadas e processadas pela ferramenta. Esse template servirá como base para a geração de textos personalizados.

Objetivo

Essa ferramenta permite criar textos padronizados com trechos personalizados de forma rápida e segura, substituindo automaticamente as variáveis indicadas no template com os valores inseridos nos campos gerados.
