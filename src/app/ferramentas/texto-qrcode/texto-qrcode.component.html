Custom Text and URL QR Code Generator

Create a downloadable QR code from text, links and other short content, then customize its colors, module style and appearance. QR codes can be used for websites, contactless sharing, labels, menus and printed materials.

Always scan-test a customized code before publishing it. Strong contrast, sufficient quiet space and an appropriate error-correction level improve reliability across cameras and print sizes.

A Bit About QR Code

A QR Code is a two-dimensional code created to store information compactly and make it accessible to readers or smartphone cameras.

It can share links, contacts, product information, payments, tickets, menus, coupons and much more.
