Instructions

How to generate passwords

Adjust the length with the slider, select the desired character sets and click Generate. Passwords are generated automatically whenever a setting changes. Use the copy button next to each password or Copy all to copy the complete list.

Character sets

A–Z and a–z include the uppercase and lowercase letters of the alphabet. 0–9 adds the numeric digits. !@#… adds special symbols. At least one set must be active. The No ambiguous option removes visually similar characters such as I, l, 1, O, 0 and o, reducing errors when typing the password manually.

Strength indicator

The estimated strength takes into account the password length and the diversity of active character sets. Passwords with 16 or more characters using all four sets are rated as Very strong. For critical accounts, at least 20 characters are recommended.

Password best practices

A secure password combines high length with real randomness. Passwords generated by cryptographic algorithms, like those produced by this tool, are much more secure than passwords created by humans, which tend to follow predictable patterns such as replacing letters with numbers (a→4, e→3) or using dictionary words.

Never reuse passwords across different services. Use a password manager to store unique and complex passwords for each account. This tool performs all generation locally in the browser, without transmitting any data to external servers.
